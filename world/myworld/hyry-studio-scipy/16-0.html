Visual使用手册

在ipython中交互式地使用

visual库可以在IPython中交互式的使用，启动ipython之后，只需要先执行：

>>> from visual import *

之后就可以随心所欲的调用visual库通过的函数。需要注意的是如果你关闭了visual弹出的场景窗口的话，ipython对话也随之结束。如果你需要关闭场景窗口可以用下面的语句：

>>> scene.visible = False
