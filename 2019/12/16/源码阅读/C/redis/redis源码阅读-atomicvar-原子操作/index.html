¶redis源码阅读-atomicvar(原子操作)

¶atomicvar介绍

  • atomicvar是redis封装的原子操作函数。其中包括了三种原子操作方法，gcc提供的內建__atomic和__sync,mutex函数族以及pthread线程库提供的mutex系列函数。
  • 原子操作主要是争对多线程下的线程安全。
