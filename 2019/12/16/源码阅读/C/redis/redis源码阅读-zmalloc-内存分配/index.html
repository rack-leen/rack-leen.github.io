¶redis源码阅读-zmalloc(内存分配)

¶zmalloc介绍

  • redis支持使用tcmalloc,jemalloc内存分配器分配内存，也支持操作系统平台提供的内存分配库。
  • zmalloc是redis封装的内存分配函数。redis将多种内存分配方法封装为统一的接口zmalloc，让其统一对外提供功能，避免跨平台不兼容问题。
