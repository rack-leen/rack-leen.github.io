¶redis源码阅读-zipmap(压缩字典)

¶zipmap介绍

zipmap,是redis定义的压缩字典，是用来保存键值对的数据结构。它保证hash表刚创建以及元素较少时，用更少的内存来存储，同时对查询效率没太大影响。
