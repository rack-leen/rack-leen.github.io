¶对文件进行读写操作

  • f.open(filename,mode) filename:文件名；mode:进行操作的方式，读操作（r）,写操作（w）,只能对文件增加数据（a）,进行读写操作（r+）
  • 用with关键字进行文件操作,在完成任务后将会关闭文件,如果不用，需要使用f.close()函数关闭文件
