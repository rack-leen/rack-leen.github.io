select-poll和epoll多路复用(事件驱动)技术

linux内核集成了select,poll和epoll三种NIO多路复用技术(多路复用使用事件驱动模型作为编程范式),多路复用技术可以一个进程同时监视多个文件描述符，在大并发条件下可以提高线程利用率，减少线程资源浪费。
